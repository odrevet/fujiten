<p>Fujiten is an offline Japanese dictionary application made with the flutter framework.</p>

<p>Definition and kanji comes from the EDICT dictionary, compiled as a database from the edict_database project.</p>

<br><big>Setup</big>

<p>In order for fujiten to work, the Expression Database and the Kanji Database are needed.</p>

<p>The databases can be downloaded and installed from fujiten via the "settings/databases" menu or by downloading the database manually from <a href="https://github.com/odrevet/edict_database">https://github.com/odrevet/edict_database</a></p>

<br><big>Top menu</big>

<b>Bars</b>

<p>Access the settings menu.</p>

<p>The settings menu allows you to download the dictionaries, set brightness and read legal information.</p>

<b>Insert</b>

<i>Radicals &lt;&gt;</i>

<p>Will match kanji composed with the selected radicals</p>

<i>Kanji character Ⓚ</i>

<p>Will match any kanji</p>

<i>Kana character ㋐</i>

<p>Will match a hiragana/katakana character</p>

<i>Joker .*</i>

<p>Any match</p>

<b>Convert</b>

<p>In case your device is not equipped with a Japanese input keyboard, fujiten can convert Latin character (romaji) to hiragana or katakana.</p>

<p>Lowercase romaji will be converted to hiragana, uppercase romaji will be converted to katakana.</p>

<br><big>Kotoba / Kanji search</big>

<p>When Kotoba is selected, fujiten will search for expression.</p>

<p>When Kanji is selected, fujiten will search for kanji.</p>

<b>Clear</b>

<p>Clear the input field.</p>

<b>Search</b>

<p>Run the search.</p>

<br><big>Tips</big>

<p>• Search are performed with regular expression, quantifiers "{}" and meta-characters like "." and others can be used</p>

<p>• Use search by radical &lt; &gt; when searching for an expression which you do not know a kanji but recognize some of its radicals example: ＜化＞</p>

<p>• When no results, add ".*" at the beginning or the end of your search</p>
